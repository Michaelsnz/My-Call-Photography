# Through The Lens 📸

## Author

Michael Mwendwa

## Description

Through The Lens is a one-page photography portfolio website showcasing
three categories of photography: automotive, landscape, and fashion.
The site features a welcome section, an about page, a curated photo
gallery, a reviews form, and a contact form — all styled with a warm
light-pink theme.This work was done through extensive research with the help ai tools such as claude,
that were used to understand how to do complex codes and styles.
PLEASE NOTE: The Code on this doccument was hand-typed by only me
and not copy pasted from anywhere else.

## Project Setup Instructions

To run this project locally:

1. Clone or download this repository
2. Make sure both `index.html` and `style.css` are in the same folder
3. Place any photos inside an `images/` folder in the same directory
4. Open `index.html` in any web browser — no installs needed!
